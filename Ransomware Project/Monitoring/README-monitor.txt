README:  monitor-ransom.exe

This file is the monitor to detect a ransomware attack and prevent it by shutting down the system

Step 1:  run monitor-ransom.exe in a command prompt. Wait at least 5 minutes as this prorgram will 
calculate checksum of all files for any future recovery.

It will monitor the system and alert in case of an attack. On getting confirmation to shutdown, system will
be shutdown
